# visualizations/

Generated by the `research-viz` skill on __TIMESTAMP__.

This folder is a self-contained workspace for parsing, plotting, and interactively exploring the dataset under `../data/` (sibling to this folder).

- See `info/how_to_use.md` for how to run things.
- See `info/context.md` for what's been done so far and where to find it.

Three entry points:

- `parse_input.sh` — run the parser to clean raw data into `intermediate_data/parsed_results.csv`.
- `generate_plot.sh` — generate static plots into `plots/<slug>/`.
- `interactive_page.sh` — launch the streamlit explorer.
